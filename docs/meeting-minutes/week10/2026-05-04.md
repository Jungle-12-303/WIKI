| 추가 메모 | `file descriptor`는 여러 syscall에서 공통적으로 사용될 가능성이 높아 조 내 이해도가 가장 높은 사람이 맡기로 했다. |

## 공통 공유 사항

### 코치님 질의 관련

- 코치님이 방문하셨고, 해결이 어려운 질문들을 모아 토요일에 별도 질의응답 시간을 가질 수도 있다는 공유가 있었다.
- 위키에 질문을 올렸음에도 해결되지 않는 문제나, 조별로 계속 막히는 이슈가 있다면 따로 모아두는 것이 좋다.
- 금요일 점심 이후 코치님께 질문 목록과 가능한 시간을 정리해 연락드릴 예정이다.
- 긴급하게 막힌 조가 있다면 따로 연락해 도움을 요청할 수 있다.

### Extension 관련

- Extension은 현재 publish 진행 중이다.
- 현재는 팀 간 GitHub 주소만 공유된 상태이며, publish가 성공하면 다시 안내받을 예정이다.
- GitHub repository에 들어가면 extension이 어떤 기능을 지원하는지 설명이 적혀 있다.
- 모든 내용을 반드시 읽을 필요는 없으며, 필요 시 각 팀을 돌면서 세팅을 도와줄 예정이다.
