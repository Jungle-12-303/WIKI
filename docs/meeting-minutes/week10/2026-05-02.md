| 안건 | 결정 상태 | 결정 내용 |

## 팀별 공유 내용

### 7조

| 항목 | 내용 |
|------|------|
| 커밋 컨벤션 | 모두 동의. 추가로 부사어를 제거하자는 의견이 나옴. 수정 의견에 대해서는 커밋 컨벤션을 참고하거나 스킬을 사용한 뒤 본인이 추가로 수정하는 방식으로 사용하는 걸로 권유하도록 결정남. |
| 협업 방법 | 현재 3명이서 페어 프로그래밍 중. 속도는 느리지만 셋의 시너지가 더 생김. 초반 2시간에서 현재는 40~50분 정도씩 돌아가며 진행 중이고, 1시간이 적당한 것 같다는 의견이 나옴. 머지 걱정이 없다는 장점이 있음. |
| AI 사용 | 함수, 주석만 AI를 활용해 이해함. 코드 작성은 AI 사용하지 않음. |
| 진도 | `load()` 함수에서 argument를 tokenize해서 stack에 올리는 부분 진행 중. |
