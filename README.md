# Jungle 12-303 WIKI

크래프톤 정글 12기 303호의 공동 학습 기록 저장소입니다.

팀원 모두가 동일한 키워드 체계 아래 학습한 내용을 정리하고, 서로의 글을 참고하며 함께 성장하는 것을 목표로 합니다.

## 구조

```
docs/
  cs-기초/           CS 기본 개념 (메모리, 포인터, 보수 표현 등)
  algorithm/         알고리즘 및 자료구조
  malloc-lab/        동적 메모리 할당기 구현
  네트워크/           TCP/IP, HTTP, REST API, OSI 7계층
  os/                운영체제 핵심 개념
  pintos/            Pintos 프로젝트 (Project 0~4)
  ai/                신경망, LLM, AI 응용기술
  db/                데이터베이스
  web-security/      웹 개발 & 보안
  팀운영/
    컨벤션/           Git, 코드, PR 규칙
    외부리소스/        개발 환경, 도구, 학습 자료
  meeting-minutes/   팀장 회의록
```

각 폴더의 `README.md`가 해당 카테고리의 목차 역할을 합니다.

## 기여 방법

[CONTRIBUTING.md](CONTRIBUTING.md)를 읽고 규칙을 따라주세요.

핵심 규칙:
- 각 키워드 파일은 한 사람이 작성합니다. 작성을 시작하면 README.md 상태를 `작성 중 (@이름)`으로 변경하세요.
- 파일명은 영문 소문자 + 하이픈: `cpu-vs-gpu.md`
- 커밋 메시지: `docs(<카테고리>): <작업 내용>`
